## Contributing

Build:

```console
mix compile
```

Test:

```console
mix test
```

```console
MIX_ENV=test mix deps.get
mix test
```

Format code:

```console
mix format
```

Generate docs:

```console
mix docs
```

Publish:

```console
mix hex.publish
mix hex.publish docs
```

This project uses the Contributor Covenant version 2.1. Check [CODE_OF_CONDUCT.md](/CODE_OF_CONDUCT.md) for more information.
